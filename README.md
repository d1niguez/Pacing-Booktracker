# Pacing 📖

A reading tracker that helps you stay on pace with your reading goals.

## Features
- Track books you're currently reading
- See your reading progress and pacing
- Calculate estimated finish dates
- Mark books as completed
- User accounts — your books are private to you

## Built With
- Python & Flask
- SQLite & SQLAlchemy
- HTML & CSS
- Flask-Login for authentication

## Live Demo
https://pacing-booktracker.onrender.com

## How to Run Locally
1. Clone the repo
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run: `python app.py`
6. Open `http://127.0.0.1:5000`
